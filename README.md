# Recipe App API

This is a RESTful API for managing recipes built using Node.js, Express, and MongoDB.

## Features

- CRUD operations for recipes
- Pagination for listing recipes
- Input validation
- Error handling

## Requirements

- Node.js v12+
- MongoDB
